# 人材管理システム (Jinzai System)

## 概要 (Overview)
Google Apps Script (GAS) と Google スプレッドシートを活用した人材紹介事業向けの管理システムです。
候補者（登録者）の管理、求人案件の管理、面接から採用（内定）までのプロセスを一元管理し、業務の効率化を図ります。日本およびミャンマー拠点のスタッフが共同で運用することを想定して設計されています。

## 主な機能 (Features)
* **候補者管理**: 「登録者マスタ」への新規登録、更新、削除。顔写真データを含んでいても高速に一括更新可能なロジックを実装。
* **案件・採用管理**: 求人案件の登録、修正、削除機能。および案件に対する候補者の「採用（内定）」登録機能。
* **自動リスト同期**: 候補者のステータス（未採用・採用）に応じて、該当シート（「未採用者一覧」「採用者一覧」）へデータを自動振り分け・同期。特定技能要件の抽出と表記統一（例: 外食業技能（合格）など）も自動化。
* **ファイルリンク連携**: 関連ファイルURLをGoogleドライブのファイル名を取得して「スマートチップ風」に自動変換。
* **履歴書・簡易リスト生成**: 候補者IDを指定して、提出用の履歴書や簡易リストを自動生成する機能。

## システム構成 (System Structure)
### スプレッドシート（データベース）
* **登録者マスタ**: 全候補者のベースデータ（ID: SD-XXXX）
* **案件管理**: 求人案件データ（ID: JOB-XXXX）
* **リスト・出力用**: 未採用者一覧, 採用者一覧, 簡易リスト, 履歴書, 面接管理
* **各種マスタ**: 事業者マスタ, 送り出し機関マスタ, 日本語学校マスタ, 企業情報, 面談用質問

### Google Apps Script (バックエンド・フロントエンド)
* `01_共通・メニュー.gs` / `JS_Common.html` : 共通処理・UIの制御
* `02_候補者管理.gs` / `HTML_Candidate.html` / `JS_Candidate.html` : 候補者データ操作
* `03_案件・採用管理.gs` / `HTML_Job.html` / `JS_Job.html` : 案件データ操作・採用登録
* `04_事業者管理.gs` / `HTML_Company.html` : 事業者データ操作
* `05_マスタ連携・その他.gs` : リストの自動同期（`syncListSheets`）、辞書生成などの補助機能
* `履歴書作成.gs` : 履歴書の自動出力処理
* `MainSidebar.html` / `Sidebar_CSS.html` : サイドバーのUIとスタイリング
